Leetcode 刷题总结 - Code546 移除盒子

发表于 2020-08-15 更新于 2021-01-13 分类于 Leetcode 阅读次数： Valine：

题目描述

给出一些不同颜色的盒子，盒子的颜色由数字表示，即不同的数字表示不同的颜色。

你将经过若干轮操作去去掉盒子，直到所有的盒子都去掉为止。每一轮你可以移除具有相同颜色的连续 k 个盒子（k >= 1），这样一轮之后你将得到 k*k 个积分。

当你将所有盒子都去掉之后，求你能获得的最大积分和。

详见：https://leetcode-cn.com/problems/two-sum-ii-input-array-is-sorted/

这道题是我刷 Leetcode 以来，每日一题中难度最高的一道题，所以在此记录下解题方法。

解法一：深度优先搜索

整理自：https://www.showsql.com/2019/12/10/leetcode546-remove-boxes/

数字 A[i] 在数组 A 中多个位置处出现。将相邻的 A[i] 连起来，这样 A[i] 在数组中，就形成了一个个连续的块。

例如：[1, 3, 3, 2, 2, 3, 3, 3, 4, 3, 1] 中数字 3，构成了 3 个块。[3,3]，[3,3,3]，[3].

那么，处理数字 3，可有如下的方法。

  1. 处理 [3,3]
    • 移除 [3,3]，然后对 [2, 2, 3, 3, 3, 4, 3, 1] 处理。
    • 保留 [3,3]，移除 [2,2]，使得 [3,3] 与 [3,3,3,4,3,1] 相邻。
  2. 处理 [3,3,3]
    • 移除 [3,3,3]，然后对 [4, 3, 1] 处理。
    • 保留 [3,3,3]，移除 [4]，使得 [3,3,3] 与 [3,1] 相邻。
  3. 处理 [3]
    • 移除 [3], 然后处理 [1]。

从上面粗略的思路中，可总结出相同数字块的处理方法。数字块之间有其它不同的数字填充。

对块 0 来说，可直接将其移除算分，块 0 之后的区域递归计算

或者，保留块 0。先将块 0 与块 1 之间区域递归计算后，块 0 与块 1 就连起来了。再递归计算块 1 和块 1 之后的区域。

对中间的块 j 来说，类似，要么与前面的块 j-1 拼接爆掉，要么保留到下个块 j+1.

同样，从最后一个块向前考虑，也能用同样的方法。

上面递归算法的特点是，区间长度从长到短，直到长度为 0.
